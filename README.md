# MATH BLASTER

Jeu arcade : des opérations tombent du haut de l’écran, tire sur la **bonne réponse** avant qu’elles touchent le bas.

## Lancer le jeu

Ouvre `index.html` dans un navigateur (double-clic ou `open index.html`).

## Modes

- **Facile** — Additions (1–20)
- **Normal** — Multiplications (tables)
- **Avancé** — Puissances (2^3…) et fractions (ex. 1/2 = ?)

## Boss

Toutes les 5 bonnes réponses, un **boss** apparaît : 3 étapes à enchaîner (12×12, 7³, racine de 169…). Bonus +500 pts à la fin.

## Contrôles

- **Souris** : viser + clic pour tirer
- **Tactile** : viser avec le doigt + relâcher pour tirer

Vies : 3. Une mauvaise réponse ou une opération qui atteint le bas enlève une vie.
